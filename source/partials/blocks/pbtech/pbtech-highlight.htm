Highlight of the solution


Add a quiz feature to help users find the right laptop

Before, on the landing page, the user is only allowed to choose one category at a time. An example would be, a user selecting 'business laptop', which will result in 381 products on the next listing page. Now, by using the quiz feature, if the user selects ‘business laptop> mac> medium size’, there will only be 23 results left.

This could help narrow down the search results, so that users can find a meaningful starting point.

Quiz reduced irrelevant results by 91 %
