Using “How Might We” Questions to Turn Insights into Opportunities

One selected example

"How might we help users select the right laptop easily?"

1. Brainstorming

I organised a brainstorming session first and collaborated with three developers.

It is better to know the feasibility at the early stage.

It also helps us to build consensus with the developers.

working as a team



2. Where the idea of a quiz came from?

We found an article on the landing page, which guides users how to select a laptop step by step.

However, none of the users discovered it, not to mention read it.

During brainstorming, I came up with the idea of converting this guide into a quiz with simple steps, because few people would read through such a long article.



3. Testing

I tested and discovered that if we designed the quiz into three steps:

Purposes ---> OS ---> Size

It would narrow down the number of search results by around 90%.



4. User flow

Before we turned it into a prototype, we made a user flow for this new feature.
